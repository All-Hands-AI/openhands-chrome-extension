OpenHands GitHub Launcher Settings

Your API key is used to authenticate with the OpenHands API. It will be stored securely in your browser.

How to get your API Key

  1. Log in to your OpenHands Cloud account
  2. Navigate to the Settings page
  3. Locate the "API Keys" section
  4. Click "Generate New Key"
  5. Give your key a descriptive name (e.g., "Chrome Extension")
  6. Copy the generated API key and paste it above
